MeetTwin — Project About

What MeetTwin Does

MeetTwin creates a lightweight, real-time 3D avatar that joins video meetings and behaves on your behalf according to rules you set. It can speak pre-approved lines, nod, react to conversational cues, and produce meeting summaries for the owner.

How it works — overview

  1. Avatar creation: Capture a short video or photo set. A pipeline generates a lightweight 3D model and expression rig optimized for browsers.
  2. Real-time proxy: The avatar joins meetings via a secure session token and streams synthesized audio & lightweight animation frames rather than raw webcam video when permitted.
  3. Summary & logs: After meetings, MeetTwin provides a concise summary and optional timestamps. Logs and audio are stored only if the user opts in.

Key features

Low-bandwidth avatar

Optimized mesh + expression blending that avoids streaming full video.

Policy-driven actions

Set rules for what the avatar can say or do; sensitive actions require confirmation.

Real-time text & audio

Optional on-device TTS and speech recognition for natural interaction.

Post-meeting summaries

Automatic meeting highlights, action items and speaker timestamps.

Privacy & ethics

MeetTwin is designed with privacy first: avatar generation happens client-side when possible, recordings are opt-in, and any automated action must conform to the owner-defined policy. We recommend clear consent from other participants before using an avatar in a meeting.

Roadmap

  • Phase 1 — Core avatar generation & browser runtime (MVP)
  • Phase 2 — Policy editor, meeting integration (WebRTC adapters)
  • Phase 3 — Natural language action rules, improved on-device models
  • Phase 4 — Enterprise features: audit logs, admin controls, SSO
